# ABC College Website

## Project Overview

ABC College Website is a responsive full-stack web application developed using Node.js, Express.js, MySQL, HTML, CSS, Bootstrap, and JavaScript. The website provides information about the college, including departments, courses, gallery, and contact details. It also includes an admin panel to manage contact messages submitted by users.

---

## Features

- Responsive design for Mobile, Tablet, and Desktop
- Home Page
- About Page
- Departments Page
- Courses Page
- Gallery Page
- Contact Form
- Contact form data stored in MySQL
- Admin Login
- Admin Dashboard
- Search Contact Messages
- User-friendly Interface
- Secure Backend using Express.js

## Installation

1. Clone the repository

```
git clone YOUR_GITHUB_REPOSITORY_LINK
```

2. Open the project

```
cd ABC-College-Website
```

3. Install dependencies

```
npm install
```

4. Import the MySQL database

- Create the database
- Import `database.sql`

5. Start the server

```
node app.js
```

6. Open

```
http://localhost:3000
```
